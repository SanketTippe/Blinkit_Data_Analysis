## Overview
This project presents an end-to-end analysis of the Blinkit dataset using **Excel** for initial exploration, **Power BI** for interactive dashboards, and **Kaggle** as the dataset source. The goal is to uncover insights into sales, customer behavior, and outlet performance, and to visualize them in a business-friendly format.

## Objectives
- Perform data cleaning and preprocessing in Excel
- Build interactive dashboards in Power BI
- Analyze sales trends, product categories, and outlet performance
- Provide actionable insights for business decision-making

## Dataset
- **Source:** Kaggle (Blinkit Sales Dataset)
- **Features:**
  - Item Identifier  
  - Item Weight  
  - Item Fat Content  
  - Item Visibility  
  - Item Type  
  - Item MRP  
  - Outlet Identifier  
  - Outlet Establishment Year  
  - Outlet Size  
  - Outlet Location Type  
  - Outlet Type  
  - Item Outlet Sales

## Key Insights
- Sales distribution varies across outlet types and sizes  
- Item visibility and MRP strongly influence sales performance  
- Customer preferences differ based on fat content and item type  
- Outlets in Tier-1 cities outperform those in smaller towns
